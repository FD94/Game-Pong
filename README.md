# 🏓 PONG Game

A simple browser-based **PONG game** built with **HTML, CSS and JavaScript**.  
The player competes against a moving machine paddle. The first to reach the winning score wins the game.

This project recreates the classic **Pong arcade game** with a modern neon UI style and background music.

---

## 🎮 Gameplay

- The player controls a paddle using the **arrow keys**.
- The machine paddle moves automatically.
- The ball bounces off paddles and top and bottom walls.
- When the ball passes a paddle, the opponent scores.
- The first player to reach the **winning score** wins the match.

---

## 🕹️ Controls

| Key          | Action           |
| ------------ | ---------------- |
| ⬆ Arrow Up   | Move paddle up   |
| ⬇ Arrow Down | Move paddle down |

---

## 🏆 Winning Condition

The first player to reach **7 points** wins the game.

After a win:

- A message is shown.
- The game resets and returns to the main menu.

## ⚙️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **DOM Manipulation**
- **requestAnimationFrame()**
- **Session Storage API**
- **Audio API**
